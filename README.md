# dotfiles

- fish shell with fisherman
- karabiner with JIS and mouse button remap

## Usage

Installing fish config

```
stow -vvt $HOME fish --ignore .DS_Store
```
## Brewfile

Install

```
brew bundle
```

Check if everything is installed
```
brew bundle check
```
